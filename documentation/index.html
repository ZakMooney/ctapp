Welcome

This site aims to be a comprehensive guide to Cucumber Tony. We'll cover topics such as getting your network set up, monitoring your access points, as well as managing and monitoring small or large WiFi networks


What is Cucumber Tony

Cucumber Tony is a cloud based WiFi platform. It gives you control over your WiFi access points.

The dashboard gives you a visual overview of all your Wi-Fi networks, making it really easy to set up and run a reliable Wi-Fi network.

You control your Wi-Fi settings from your dashboard. Setting up Wi-Fi, creating splash pages, security settings and usage reports.

You see everything that's happening - see who's connected to your Wi-Fi network in Edinburgh when you're holidaying in Barcelona.
