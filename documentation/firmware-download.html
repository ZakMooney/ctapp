Firmware Download

If you are looking third party integration, please see the previous step.

This section covers devices that can be flashed with our firmware.

The devices that we have firmware support for are as follows:

  • Ubiquiti UniFi UAP
  • Ubiquiti UniFi Pro
  • Ubiquiti UniFi LongRange
  • Ubiquiti UniFi Outdoor
  • Ubiquiti UniFi Outdoor +
  • TP-Link WDR4300
  • Engenius ESR900
  • OpenMesh

Select the box you have from above and follow the guide to install our firmware on it. Once you have done this, you can add your box to Cucumber Tony by following this step.
