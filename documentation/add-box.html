Add Your Access Point

Adding a new box to your location is a very simple process, just follow these quick steps:

  1. Select the location you want to add a new box to.
  2. Click on Boxes in the left hand side bar
  3. Click on the New box button.
  4. You will be presented a number of options to help you decide how best to add your box into Cucumber Tony. Tick one of the boxes, and then click on next. Your choices are as follows:

    5. I have a Cucumber Tony box

    This is the option for you if you either bought a box with preloaded firmware from us, or if you already have a box with our firmware installed.

    Clicking this option will allow you to use the auto detect feature.

    However, if you do not have an internet connection to the Access Point at that time, or other such issues, you will not be able to use the autodetect feature, but you will be able to manually add your box’s mac address

    I want to install the firmware

    This is the option for you if you have a supported box and want to get it running Stinky Panda.

    Clicking this option will take you to the firmware download page where you can retrieve the firmware to flash onto your Access Point.

    I have an order number or mac address

    This is the option for you if you have the mac address or Order ID ready with you, as you can use both of these to add the box into Cucumber Tony.

    Clicking on this option will open two fields for you, one for the mac address and one for the Order ID, either of which can be submitted to add it to your location.

    I have a Meraki, Ruckus or other device

    If you are trying to add a different third party device, then please make use of this option.

    Clicking on this option will open two fields for you, one for the mac address and one for the Order ID, either of which can be submitted to add it to your location. It will also list the box as a non-polkaspots box.

    I don't have a box yet

    If you don’t currently have a box for yourself that is compatible, then clicking this option will take you to the store where you can purchase all of the Access Points you want.

    Once you have your basic configuration set up and your boxes added, you can move on to tweaking it exactly how you want and trying out some more advanced features/configurations.
