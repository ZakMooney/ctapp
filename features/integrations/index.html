Cucumber Integration Features

Manage and monitor your devices from the cloud with these features.

API

A single, unified RESTful API brings the control to you. Integrate with your own systems. Control access, reboot access points, download reports.

Anything that can be done in the dashboard (and more) can be done using the API.

MailChimp

Integrate with MailChimp for your email capture, and make use of our extra special MailChimp single opt-in feature.

Campaign Monitor

Set up Campaign Monitor to work with your Cucumber dashboard for all your email capture needs.

SendGrid

Configure your email capture to work with SendGrid in a matter of minutes.

Twilio

Integrate with Twilio to set up SMS alerts and updates from your Cucumber dashboard.

Slack

Integrate with the team communication app Slack to send yourself messages on configured events.

IFTTT & Zapier

Set up compatibility with the apps you use, automate tasks and get Cucumber involved with them all.
